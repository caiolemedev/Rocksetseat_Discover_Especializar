API no Front-End com Axios

Conhecendo o Axios

Ferramenta para fazer requisições em APIs usando os verbos mais comuns. Usamos o script via CDN para utilizar a ferramenta sem precisar instalar nada via node.

Rodando API

Para o projeto usaremos o node-api-discover-main da mesma forma que fizemos no projeto anterior. Então neste passo devemos baixar, instalar e rodar o projeto na nossa máquina.

GET

Usamos o método get do Axios, similar ao Fetch feito anteriormente para exibir a resposta da API. Para que seja exibido no front precisamos transformar a resposta em texto simples usando JSON.stringfy().

POST

Usamos o método post do axios, que é muito similar ao fetch (mesma estrutura). Usamos a mesma estrutura de objeto para um novo usuário e depois passamos como parâmetro da função para ser adicionado à API.

GET com parâmetros

Usamos o método get com parâmetros de id sendo passados para a função, da mesma forma que fizemos com o fetch. Podemos exibir cada dado do usuário em uma tag html pelo id delas.

PUT

Assim como os anteriores, o put no axios funciona de forma muito similar ao fetch, onde passamos o id e os dados de usuário que queremos alterar. A construção do objeto e da função é exatamente a mesma, podemos alterar o id da função getUser para mostrar o usuário que alteramos.

DELETE

De forma muito similar ao delete do fetch, passamos para a função o id do usuário que queremos deletar e o mesmo será excluído da lista de usuários da API. Podemos observar em tempo real a atualização no array impresso no front com liveserver.

Javascript Assíncrono e Promisses

Introdução

Curso falará sobre sincronismo no JS.

Síncrono vs Assíncrono

Sistema síncrono segue um fluxo linear de tarefas, onde a anterior deve ser finalizada antes da seguinte iniciar, este é o padrão do JS. Assíncronos são processos em paralelo que são executadas ao mesmo tempo.

Callback Conceito

Funções aceitam qualquer tipo de dado, desde número até array e outras funções como argumento. Callback nada mais é que uma função passada para outra como arggumento para ser executada posteriormente.

Promise Intro

Vamos abordar nas próximas aulas o assunto promise.

O que é uma Promise

A promessa de que algo vai acontecer, ou será executada no caso. Exemplo uma corrida do uber, quando pedimos ela fica pending, aceitando seria fullfiled e rejeitada o status seria rejected e quando concluímos (com sucesso ou não) ela seria settled. A Promise é um objeto JS de que algo será realizado, é usado para operações assíncronas para não parar a aplicação.

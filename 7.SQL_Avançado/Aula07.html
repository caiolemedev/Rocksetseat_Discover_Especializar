Especializar 7.7 - SQL Avançado - Comando nas Tabelas

CREATE TABLE

Para criarmos um banco de dados primeiro criamos um arquivo escola.sqlite e abrimos no Beekeeper. Iniciamos com o comando a seguir para criar a primeira tabela com os campos configurados conforme descrição. Usamos o campo AUTOINCREMENT para garantir que o campo seja único e seja criado automaticamente quando adicionarmos um novo aluno. Exemplo: CREATE TABLE alunos (matricula INTEGER PRIMARY KEY AUTOINCREMENT, nome TEXT, cpf INTEGER UNIQUE, responsável TEXT). Ao rodarmos o comando é gerada uma tabela com os campos em branco, fazemos o mesmo para a tabela professores. CREATE TABLE professores (id_professor INTEGER PRIMARY KEY AUTOINCREMENT, nome TEXT, cpf INTEGER UNIQUE, materia TEXT). E criamos também a tabela de relação chamada aulas: CREATE TABLE aulas (id_professor INTEGER, matricula INTEGER, FOREIGN KEY (id_professor) REFERENCES professores(id_professor), FOREIGN KEY (matricula) REFERENCES alunos(matricula)). Os comando de criação não retornam resultados, somente criam as tabelas.

ALTER TABLE

Podemos usar este comando para alterarmos nome da tabela, nome de campos usando o seguinte comando: ALTER TABLE aulas RENAME COLUMN id_aluno TO matricula_aluno.

DROP TABLE

Este comando serve para excluirmos uma tabela, por exemplo: DROP TABLE alunos, fazendo todo o registro e a tabela deixar de existir por completo.

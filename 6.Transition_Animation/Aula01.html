Transition and Animation

Abertura

Boa vindas ao CSS transition e animation para mudar a forma e manipular elementos.

O que é CSS transition?

CSS transition permite manipularmos elementos CSS de forma que a mudança de propriedade seja suave ou com animações específicas. Só pode ser usada com propriedades que são quantificáveis, como cor, padding, marging, oppacity, mas não podemos usar no display block pra none por exemplo.

Transition property e duration

Na prática podemos ver que a transition funciona para o estado que estamos indo, portanto se quisermos uma volta suave precisamos adicionar transition no estado inicial. Precisamos adicionar ao transition-property todas as propriedade que queremos transicionar de forma suave, se deixarmos o width fora ele será feito instântaneamente, podemos simplesmente usar all como parâmetro e todas as propriedades alteradas terão a transição suave.

Transition timing funcions

Essas são tipos de animação, no arquivo transition testamos algumas como ease, linear, ease-in, ease-out, ease-in-out e cube. A particularidade do cube é que ela pode ultrapassar os limites pré-estabelecidos e voltar, como um estilingue.

Transition delay

O transition delay é uma propriedade que irá adicionar um tempo para que a transsição inicie, ou seja, atrasa o tempo de aplicação.

Transition property all

Usando o property all podemos alterar todas as propriedades do elemento de uma única vez, aplicando o timing e o delay.

O que é CSS animation?

Animation movimenta elementos CSS na página sem a necessidade de interação do usuário.

Animation name e duration

Para fazermos a animação usamos a propriedade @keyframes do CSS e dentro dela podemos alterar as propriedades que desejamos alterar por exemplo from (ponto de partida) e to (ponto de chegada). Quando a animação acada todas as propriedades do objetos são restauradas ao original.

Keyframes com porcentagens

Em vez de usarmos o from e to, podemos usar porcentagens para determinar a evolução da animação.

Animation delay

O delay é o atraso para que a animação se inicie após x sgundos depois da página carrgar.

Animation iteration count

Esta propriedade permite fazermos repetição da animação. O delay só funciona para o início da animação, as demais repetições não terão delay. Podemos usar o infinite para que a animação seja repetida o tempo todo.

Animation direction

Podemos alterar a direção da animação, por exemplo aplicando o reverse, alternate e alternate-reverse.

Animation timing function

Da mesma forma que o transition no animation podemos usar timing functions para alterar o tipo de animação, mais rápida no início ou no final, etc.

Animation fill mode

Podemos usar o animation-fill-mode para determinar configurações extras na animação, como fixar as propriedades do final da animação, fixar as propriedades do delay ou ambos os casos. As referências sempre serão a primeira e última chaves.

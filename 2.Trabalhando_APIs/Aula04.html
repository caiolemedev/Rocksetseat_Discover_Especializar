Consumindo APIs com NodeJS

API do github

Apresentando a api do github que possui biblioteca de emojis, usuários, etc. Os dados de usuário apresentados são dados públicos, por exemplo foto, nome, url do perfil, repositórios. Para usar dados não públicos precisamos de uma autenticação do github, por exemplo para visualizar repositórios privados ou realizar alguns outros verbos como POST, PATCH e DELETE. Nesta aula vamos testar somente com os dados públicos. Usamos o Insomnia para testar uma requisição GET de dados dos usuários.

Consumindo com Axios

Usando o módulo do axios podemos usar dentro dele o método get para acessar uma api na internet, no caso, o github. Utilizamos o template literals ou template string para transformar o retorno da url da imagem de perfil na própria imagem via tag img. Observando no insomnia que nosso servidor local que criamos pode consumir apis sem problemas.
